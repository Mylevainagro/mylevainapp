

CLARIFICATION MÉTIER – NOTION DE PLACETTE ET LOGIQUE D’OBSERVATION

OBJECTIF

Corriger l’interprétation actuelle de l’app concernant les observations terrain.

Le point clé à comprendre est le suivant :

👉 on n’observe pas forcément un rang entier
👉 on observe souvent une placette
👉 une placette est une petite zone fixe et répétable d’observation dans une parcelle expérimentale



1. DÉFINITION DE LA PLACETTE

Définition simple

Une placette est un sous-ensemble précis de la parcelle, utilisé pour faire des observations répétées dans le temps.

Dans notre cas, en vigne, une placette peut correspondre par exemple à :

7 ceps
10 ceps
ou un petit groupe fixe de pieds
👉 Ce groupe doit être toujours le même entre T0, T1, T2, etc.



Pourquoi une placette ?

On utilise une placette pour :

éviter d’observer toute la parcelle de manière floue
standardiser les observations
comparer toujours la même zone
rendre les données scientifiquement comparables
permettre une répétition expérimentale


2. DIFFÉRENCE ENTRE RANG, MODALITÉ ET PLACETTE

Rang

Le rang = une ligne entière de vigne.

Modalité

La modalité = le traitement ou la condition appliquée.

Exemples :

M0 = témoin
M1 = levain 1/4
M2 = levain 1/2
Placette

La placette = la petite zone observée dans une modalité.



Exemple concret

Parcelle : Merlot 3
Protocole : P1
Modalités :

M0 = témoin
M1 = levain 1/4
M2 = levain 1/2
Pour la modalité M1, on peut avoir :

Placette 1 = 7 ceps situés au début du rang
Placette 2 = 7 ceps au milieu du rang
Placette 3 = 7 ceps à la fin du rang
👉 On observe ces mêmes placettes à chaque passage.



3. CONSÉQUENCE POUR L’APP

L’observation ne doit pas être centrée sur le rang uniquement

Aujourd’hui, si l’app raisonne en “rang”, ce n’est pas assez précis pour la logique expérimentale demandée.

Il faut pouvoir saisir :

Site
Parcelle
Modalité
Placette
Répétition éventuelle


Nouveau principe

Pour une observation :

on observe une placette rattachée à une modalité

Pour un traitement :

on peut traiter une modalité sur un rang entier ou sur une zone définie



4. CE QUE DOIT PERMETTRE L’APP

Dans la page “Nouvelle observation”

Remplacer la logique actuelle basée sur “Rang” par une logique plus précise :

Identification

Site
Parcelle
Modalité
Placette
Date
Heure


Champ Placette

Ajouter un champ dédié :

placette_id
ou
placette_nom
ou
numero_placette


Description possible d’une placette

Une placette peut contenir :

numéro de placette
nombre de ceps
position dans le rang
numéros des pieds marqués
commentaire libre


5. EXEMPLE DE SAISIE ATTENDUE

Exemple 1

Site : Château de Piote
Parcelle : Merlot 3
Modalité : M1
Placette : Placette 2
Observation faite sur : 7 ceps marqués


Exemple 2

Site : Pape Clément
Parcelle : Cabernet 1
Modalité : M0
Placette : Placette 1
Observation faite sur : pieds 10 à 16


6. POUR LE RENDEMENT

La notion de placette est aussi essentielle pour les mesures de rendement.

Les champs rendement doivent être reliés à la placette observée, pas seulement à la parcelle générale.

Exemples :

nombre de grappes sur les ceps de la placette
poids des grappes de la placette
poids de 100 baies de la placette
rendement estimé à partir de la placette


7. POURQUOI C’EST IMPORTANT SCIENTIFIQUEMENT

Si on ne distingue pas la placette :

on mélange des zones différentes
on perd la répétabilité
on fragilise la comparaison entre dates
on rend les données moins exploitables pour la science et pour l’IA
Si on intègre bien la placette :

on observe toujours la même unité expérimentale
on peut comparer T0 / T6 / T12
on renforce fortement la qualité des données

9. RÈGLE MÉTIER À RESPECTER

Traitement

Le traitement peut concerner :

une parcelle entière
un rang
une modalité
Observation

L’observation doit concerner :

une modalité
une placette précise
👉 Donc :
traitement ≠ observation
rang ≠ placette
modalité ≠ placette



10. RÉSUMÉ ULTRA CLAIR

Une placette est :

une petite zone fixe d’observation
représentative d’une modalité
suivie dans le temps
indispensable pour les mesures fiables
Dans l’app, la saisie d’observation doit donc se faire sur :

Site
Parcelle
Modalité
Placette
et non pas uniquement sur :

Site
Parcelle
Rang


11. CONSIGNE POUR LA REFONTE

Objectif :

cohérence scientifique
observations répétables
meilleure qualité des données
meilleure base pour la future IA
